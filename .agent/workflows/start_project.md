---
description: Uruchamianie projektu (Backend + Frontend)
---

Projekt wymaga uruchomienia dwóch procesów równolegle: serwera backendu i serwera deweloperskiego frontendu.

1. Otwórz pierwszy terminal (Backend):
   ```bash
   node server.js
   ```
   *Powinieneś zobaczyć komunikat: "Server running on http://localhost:3001"*

2. Otwórz drugi terminal (Frontend):
   ```bash
   npm run dev
   ```
   *Powinieneś zobaczyć link do aplikacji, np. "http://localhost:5173"*
